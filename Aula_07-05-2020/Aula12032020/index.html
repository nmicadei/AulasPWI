Tabela: funções das tags html

Tag Função
h1, h2, h3, h4, h5 Título
p Parágrafo, quebra linha
b Negrito da versão 4 do html
br/ Quebra linha
strong Negrito da versão 5 do html
i Deixa o texto em itálico
hr/ Cria uma linha
ul Cria uma lista
li Coloca os itens de uma lista
ol Cria uma lista ordenada
!-- Cria um comentário
menu Menu com uma lista de itens
em Maior ênfase em italico
cite Uma sitação
u Texto sublinhado
tt Fonte semelhante a maquina de escrever
pre Texto pré formatado
strike Risca o texto
sub Texto subscrito
center Centraliza o texto
font Altera tamanho, cor e tipo da fonte
dfn Formatação do texto definido como termo de um glossário
samp Texto de amostra
var Uma variavel ou espaço reservado para outro valor

Atributos:


Id: É uma forma de identificar um elemento, e deve ser ÚNICA para cada elemento.

Style: Permite personalizar a tag, como as cores.

Class: É uma forma de identificar um grupo de elementos. Através delas, pode-se atribuir formatação a VÁRIOS elementos de uma vez.
